# Local development

## Testing

```
npm install lerna nodemon nvm ts-node --global
lerna bootstrap --npm-client=yarn
```

Run tests:

```
yarn run test
```

Run backend API:

```
yarn run dev
```

Run frontend in dev mode: (will work if the backend is started)

```
yarn run start
```
## API

```
yarn run dev
```
Then go to http://localhost:3700/docs
